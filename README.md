# Nocoiner

Nocoiner is a **low-code app platform** for building lightweight trading utilities and internal-style dashboards quickly. This repository includes an **example implementation for trade tracking and analysis**, showing how to model trades, capture updates, and present analysis views with minimal custom code.

The goal is to make it easy to assemble practical, data-driven tools (like trade journals, trackers, and performance summaries) without the overhead of a large framework or heavy UI complexity.

## What’s included (example app)
- **Trade tracking**: record entries/exits, size, timestamps, fees, notes, and tags
- **Analysis views**: basic breakdowns and summaries for reviewing performance over time
- **UI pages** rendered with templates for fast iteration and simple customization

## Platform highlights
- Low-code friendly structure: add/adjust features by editing a small number of well-scoped modules and templates
- Opinionated, minimal UI for internal tooling-style workflows
- Designed to stay simple to run locally and straightforward to deploy

## Tech stack
- **JavaScript** (primary UI behavior and client-side logic)
- **Python** (backend / data handling)
- **Jinja** (templating)
- **HTML/CSS** (layout and styling)
- **Docker** (optional containerized running/deployment)
